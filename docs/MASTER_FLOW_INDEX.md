## Legend for Diagrams

*   **Maker**: The person starting the task (e.g., Accountant).
*   **Checker**: The manager approving the task.
*   **System**: The software processing the rules.
*   **Solid Line**: Direct Action.
*   **Dotted Line**: Reply or Notification.
